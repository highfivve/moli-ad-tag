prebidOutstreamRenderer#

â–¸ ConstprebidOutstreamRenderer(domId: string, config?: Partial<{ adText: string ; allowVpaid: boolean ; autoPlay: boolean ; height: number ; maxAllowedVastTagRedirects: number ; mute: boolean ; preload: boolean ; vastTimeout: number ; width: number }>): IRenderer

Renderer for Moli's built in outstream player.

Usage is like this:

adUnit: {
mediaTypes: {
video: {
renderer: prebidOutstreamRenderer('prebid-adslot')
}
}
}

If needed, values can be overriden like this:

renderer: { ...prebidOutstreamRenderer('prebid-adslot'), url: 'another-url.net' }

Parameters:#
