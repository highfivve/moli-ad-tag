advertiserIdKey​

readonly advertiserIdKey: string

Key within the moli config keyValues in which the advertising id can be found.

Used to infer the device_id parameter.

device_id: Optional. Mobile identifier (IDFA or ADID). In lower case. This field can be omitted if it is not possible to obtain the identifier.
