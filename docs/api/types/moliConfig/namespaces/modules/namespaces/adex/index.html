AdexList​

AdexList: object

Adex lists are not really lists. They consist of objects with the list items as keys, and the literal 1 as value:

Index Signature​

[key: string]: AdexListObject

Example​

{
     *   "Automotive": 1,
     *   "Oldtimers": 1,
     *   "Car Repair": 1
     * }
