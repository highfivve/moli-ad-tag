Interface: IConfigurable

Base interface for all states.

Extends​

  • IState.WithRuntimeConfiguration.WithModulesConfigurable

Properties​

config?​

readonly optional config: undefined

The config is undefined in the initial state of the ad tag

initialize​

initialize: boolean

If set to true, initializes the ad tag as soon as the ad configuration has been set. If set to false, nothing will initialize until moli.initialize is called

modules​

readonly modules: IModule[]

Contains the list of modules that need to be initialized

Inherited from​

WithModulesConfigurable.modules

runtimeConfig​

readonly runtimeConfig: MoliRuntimeConfig

Changeable configuration if other settings have been pushed into the que. This configuration is mutable by definition.

Inherited from​

WithRuntimeConfiguration.runtimeConfig

state​

readonly state: "configurable"

Overrides​
