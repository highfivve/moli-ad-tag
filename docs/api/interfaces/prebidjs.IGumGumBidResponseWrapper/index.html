Interface: IGumGumBidResponseWrapper

prebidjs.IGumGumBidResponseWrapper

The bidResponse.ad property contains this value if it's considered a "wrapper" creative.

Properties

auid

Readonly auid: number

Stands for ad id and contains the format delivered.

  • 59 = in-screen cascade (former mobile skin)
  • 39 = in-screen expandable (mobile expandable)
