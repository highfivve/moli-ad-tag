Prebid.js defines ‚Äútypes‚Äù of native ad for you as a convenience. This way you have less code to maintain, that is hopefully more descriptive of your intent.

For now there is only the image type, but more will be added.

The image native ad type implies the following required fields:

  • image
  • title
  • sponsoredBy
  • clickUrl

And the following optional fields:

  • body
  • icon
  • cta

see http://prebid.org/dev-docs/show-native-ads.html#pre-defined-native-types
