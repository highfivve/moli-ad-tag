The Adex

The Adex is a Data Management Platform (DMP). Their script supports regular and SPA web integrations.

Using the following mapping definition, you could transform a channel key/value into a format The Adex understands:

// mapping definition{ adexValueType: 'string', key: 'channel', attribute: 'iab_cat' }
// input key/value from moli targeting{ channel: 'Education' }
// the adex compatible mapped output{ iab_cat: 'Education' }

Integration#

In your index.ts import the Adex module and register the module.

import { AdexModule } from '@highfivve/module-the-adex-dmp';moli.registerModule(new AdexModule({  mappingDefinitions: [{ adexValueType: 'string', key: 'channel', attribute: 'iab_cat' }],  adexCustomerId: '1234',  adexTagId: '1337',  spaMode: false // non-spa web project}, window, logger));
