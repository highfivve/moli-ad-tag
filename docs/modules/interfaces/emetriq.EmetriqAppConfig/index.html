Interface: EmetriqAppConfig

emetriq.EmetriqAppConfig

InApp tracking configuration

See

https://doc.emetriq.de/inapp/api.html#get-/data

Hierarchy​

  • IEmetriqModuleConfig

    ↳ EmetriqAppConfig

Properties​

additionalIdentifier​

• Optional Readonly additionalIdentifier: EmetriqAdditionalIdentifier

Configure additional identifiers

See

https://doc.emetriq.de/#/profiling/identifiers

advertiserIdKey​

• Readonly advertiserIdKey: string

Key within the moli config keyValues in which the advertising id can be found.

Used to infer the device_id parameter.

device_id: Optional. Mobile identifier (IDFA or ADID). In lower case. This field can be omitted if it is not possible to obtain the identifier.

appId​

• Readonly appId: string

App id of app store

Example

`de.emetriq.exampleApp`

customKeywords​

• Optional Readonly customKeywords: Omit<EmetriqParams, "sid">

Additional parameters (i.e. hardfacts), which could be provided from a partner. (i.e. gender=frau&age=25)

See

https://doc.emetriq.de/#/inapp/integration

customMappingDefinition​

• Optional Readonly customMappingDefinition: EmetriqMappingDefinition[]

Optional mapping definitions. Map values from the key-value targeting map to a custom parameter that is sent to emetriq.

Inherited from​

IEmetriqModuleConfig.customMappingDefinition

linkOrKeyword​

• Readonly linkOrKeyword: EmetriqAppKeywordOrLinkConfig

At least one of the config properties link or keywords must be set.

login​

• Optional Readonly login: EmetriqLoginEventConfig

Optional configuration for login events

See

https://docs.xdn.emetriq.de/#event-import

Inherited from​

IEmetriqModuleConfig.login

os​

• Readonly os: "android" | "ios"

inApp configuration Required parameter for app tracking

sid​

• Readonly sid: number

syncDelay​

• Optional Readonly syncDelay: SyncDelay

Defines a delay for the user-sync

  • pbjs (recommened) uses the prebid.js auctionEnd event to fire the user sync.
  • number delay in ms before the script is loaded. Use this if prebid is not available

Default

if not set, there is no delay

IEmetriqModuleConfig.syncDelay
