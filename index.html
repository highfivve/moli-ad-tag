The Publisher Ad Tag

An open ad tag for publishers

Get Started

Easy to Use

Create your ad tag configuration, build the ad tag and put it along with the ad slots on your page. Done. Avoid bugs by using typescript and leverage the fully typed API.

Maximize competition

Use Google Ad Manager, Prebid.js, Prebid Server and Amazon TAM to maximize your revenue.

Customize everything

Define when and where an ad slot should be loaded. Select the proper sizes based on flexible media queries and label conditions. A module system allows you to extend your ad request pipeline.

Is this for me?

You are a publisher and want to take control of your monetization. Along with some IT folks this ad tag requires

  • Google Ad Manager as an ad server
  • a TCF 2.0 compliant CMP

What is supported?

Google

Google AdX and Open Bidding provide a strong foundation are come directly with your Google Ad Manager

Prebid.js

The defacto standard for client side header bidding. Types for popular bidders are provided to ensure easy configuration.

Prebid Server

The open source server side header bidding implementation. Removes the burden from the client side to do all the heavy lifting. This requires prebid server instance.

Amazon TAM

The Amazon transparent ad marketplace. A server side header bidding solution like Google Open Bidding or Prebid Server.
